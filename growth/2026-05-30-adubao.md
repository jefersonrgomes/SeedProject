## Evolução 2 — Hipocótilo + Site Atualizado

### Canvas: hipocótilo emergindo

**Elemento:** hipocótilo (caule embrionário)
- Posição: `cx=24, y=28-29` — dentro e logo acima da rachadura
- Cor: `#a0b840` (amarelo-verde pálido = etiolado = cresceu sem luz)
- Ponta: `#c4d850` (mais brilhante — ponta de crescimento ativa)
- Curvatura sutil: `px(23,29,P.hypo,0.3)` — o hipocótilo curva em busca da luz
- Animação: pulso lento `Math.sin(t*0.025)` — respira, está vivo

**Por que essa cor?**
O hipocótilo etiolado (que cresceu no escuro) é amarelo-esverdeado, não verde vibrante. A clorofila só se forma quando há luz. Usar `#a0b840` em vez de `#68a030` (verde-broto) é biologicamente correto e cria uma transição visual convincente: quando o broto emergir na luz, mudará para verde.

**Função adicionada:**
```javascript
function drawHypocotyl() {
  const pulse = Math.sin(t * 0.025) > 0;
  px(24, 29, P.hypo,    pulse ? 0.9 : 0.7);
  px(24, 28, P.hypoTip, pulse ? 0.5 : 0.3);
  px(23, 29, P.hypo,    0.3);
}
```
