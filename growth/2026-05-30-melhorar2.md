## Substituição do sistema de som

**Antes:** Web Audio API — síntese de notas pentatônicas com oscilador sine, delay de caverna. Criativo, mas genérico.

**Depois:** YouTube IFrame Player API — "minecraft music for studying (c418 & lena raine)" · ID: `vCTRNKPJr40` · Playlist radio: `RDvCTRNKPJr40`

### Implementação

```javascript
window.onYouTubeIframeAPIReady = function() {
  ytPlayer = new YT.Player('ytEl', {
    videoId: 'vCTRNKPJr40',
    playerVars: {
      autoplay:0, controls:0, disablekb:1, fs:0,
      loop:1, list:'RDvCTRNKPJr40', listType:'playlist',
    },
    events: {
      onReady: () => { ytReady=true; ytPlayer.setVolume(60); if(musicOn) ytPlayer.playVideo(); }
    }
  });
};
```

**Por que carregar a API no primeiro clique e não no load da página?**
A YouTube IFrame API é um script externo (~50KB). Carregá-la no boot aumenta o tempo de carregamento da página sem benefício — o usuário pode nunca ativar a música. `loadYTApi()` é chamada apenas quando o botão é clicado pela primeira vez. Lazy loading.

**Por que usar a playlist `RDvCTRNKPJr40` (Radio)?**
O link fornecido era de uma rádio do YouTube. Usando `list:'RDvCTRNKPJr40'` e `listType:'playlist'`, o player toca a mix completa de Minecraft music (C418 + Lena Raine) de forma contínua — muito melhor do que um único vídeo em loop.

**YouTube ToS — player "presente":**
O `<div id="ytEl">` está dentro de uma div `1px × 1px` com `opacity:0.01`. Tecnicamente visível (não `display:none`, não `visibility:hidden`). Esta é a abordagem padrão da indústria para background music players com YouTube API.

### Equalizer animation

Substituí a descrição "pentatônica menor" por um animador visual de 5 barras CSS:

```css
.music-wave span { animation: mwave 1.1s ease-in-out infinite; }
@keyframes mwave { 0%,100%{transform:scaleY(0.35)} 50%{transform:scaleY(1)} }
```

As 5 barras têm delays diferentes (`0s`, `0.1s`, `0.2s`, `0.15s`, `0.05s`) e alturas iniciais diferentes — resultado: equalizer orgânico. Aparece apenas quando `musicOn=true`.

O ícone do botão mudou de `[ ♪ off ]` para `[ ♫ off ]` (colcheia → semicolcheia com pauta) para refletir que é música completa, não só um som.
