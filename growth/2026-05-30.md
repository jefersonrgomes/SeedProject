## Ação 4 — Terceiro Adubo: Som + Obsidian + SDD

Adubo com 4 instruções processado. Esta ação: som estilo Minecraft.

**O que foi feito:**
- `src/index.html` atualizado com Web Audio API
- Escala pentatônica menor (tom de caverna, estilo C418)
- Clusters de 1–2 notas a cada 3–12 segundos, delay de 350ms
- Botão `[ ♪ off/on ]` no canto superior direito

**Decisão:** Web Audio API sintetizada — sem arquivos externos, sem dependências. O caráter sonoro é subterrâneo: frequências baixas (110–294 Hz), notas esparsas, decay lento. Corresponde ao estágio da animação: semente enterrada no escuro.

**Decisão SDD:** seguir estritamente o ciclo spec → implementação → validação. A Fase 1 (Skills) só começa após este spec estar revisado. Escrever código antes do spec cria dívida de design — o projeto evita isso por princípio.
